# 如何找学术论文
## [首推Google scholar](https://scholar.google.com/)
- 可根据时间线、关键词搜索
- follow 学术大牛
- 关键词为大领域时，看好文章（顶会、顶刊）。小领域或紧相关工作时，主要看小同行们现在都在哪个方向努力，哪种技术路线热门。

## [从数据集找论文（papers with code）](https://paperswithcode.com/)
- 查询最新公开数据集的最新文章

## follow 重点课题组
