给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

请注意 ，必须在不复制数组的情况下原地对数组进行操作。

 
```
示例 1:

输入: nums = [0,1,0,3,12]
输出: [1,3,12,0,0]
示例 2:

输入: nums = [0]
输出: [0]
 ```

提示:

- 1 <= nums.length <= 104
- -231 <= nums[i] <= 231 - 1
 

进阶：你能尽量减少完成的操作次数吗？

方法一：把 nums 当作栈

cpp:
```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        int stack_size = 0;
        for (int x : nums) {
            if (x) {
                nums[stack_size++] = x; // 把 x 入栈
            }
        }
        fill(nums.begin() + stack_size, nums.end(), 0);
    }
};
```

python:
```python
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        stack_size = 0
        for x in nums:
            if x:
                nums[stack_size] = x  # 把 x 入栈
                stack_size += 1
        for i in range(stack_size, len(nums)):
            nums[i] = 0
```
```python
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        stack_size = 0
        for x in nums:
            if x:
                nums[stack_size] = x  # 把 x 入栈
                stack_size += 1
        nums[stack_size:] = [0] * (len(nums) - stack_size)
```


方法二：双指针+交换元素

cpp:
```cpp
class Solution {
public:
    void moveZeroes(vector<int>& nums) {
        auto slow =0 ;
        for (auto & x : nums){
            if (x){
                swap(x,nums[slow]);
                slow++;
            }
        }
    }
};
```

python:
```python
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        slow = 0
        for fast in range(len(nums)):
            # 当 fast 找到一个非零数时
            if nums[fast] != 0:
                # 就和 slow 指向的位置交换
                # 如果 slow 和 fast 相等（比如数组开头都是非零数），就相当于自己和自己交换，没有影响
                nums[slow], nums[fast] = nums[fast], nums[slow]
                # 交换后，slow 指针前进一步
                slow += 1

```
